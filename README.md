# Electric Vehicle Analytics Dashboard

## Overview

This project is a **Streamlit web application** that integrates multiple **Tableau dashboards** to analyze electric vehicle trends, charging infrastructure, and EV model efficiency.
The application provides a simple login interface and allows users to explore multiple dashboards from a single interface.

The dashboards are embedded from **Tableau Public** and displayed inside the web application.

---

## Features

* Login system for accessing dashboards
* Interactive dashboard selection
* Embedded Tableau dashboards
* EV analytics visualization
* Custom background UI
* Streamlit based web interface

---

## Dashboards Included

1. **EV Market & Charging Infrastructure Analysis**
   Insights into EV brands, models, charging stations, and price vs range analysis.

2. **EV Charging Behavior & Infrastructure Planning Insights**
   Analysis of charging station distribution and infrastructure planning.

3. **Comparative EV Model Efficiency Dashboard**
   Comparison of EV models based on efficiency, cost, and performance.

4. **EV Cars in India (Story Page)**
   Story-based visualization explaining EV adoption and trends.

## Installation

Clone the repository

```
git clone https://github.com/your-username/ev-dashboard-project.git
```

Move to project folder

```
cd ev-dashboard-project
```

Install dependencies

```
pip install -r requirements.txt
```

Run the application

```
streamlit run tableau.py or use this website: https://evdashboardbytableau-pquy4osnhcpfbkwuuhh3ms.streamlit.app/
```
